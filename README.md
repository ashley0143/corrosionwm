# corrosionwm

Blazing fast wayland compositor written in rust

## Run

Run using `cargo run --release`

## Default Keybindings

The default keybindings are as follows:

| Keybinding | Action |
|------------|--------|
| `Left Alt + Enter` | Open terminal |
| `Left Alt + H` | Open `wofi` |
| `Left Alt + Q` | Quit |
